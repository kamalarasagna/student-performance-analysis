
# Student Performance Analysis System

## Description
A menu-driven Python application that manages and analyzes student performance data using SQLite database.

## Technologies Used
- Python
- SQLite
- SQL

## Features
- Add and view student records
- Calculate total marks, average, and grades
- Store and retrieve data using SQLite
- Perform SQL operations like CREATE, INSERT, SELECT

## How to Run
1. Install Python
2. Download or clone the repository
3. Run the program:
   python student_performance.py
